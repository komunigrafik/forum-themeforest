HTML Structure

This template using Twitter Bootstrap 3.2.0 Framework. All the information within main content is nested within a div with id- name "content". If you need more information you can read Bootstrap documentation

Below is a file structure:

  • index.html
  • category.html
  • detail.html
  • post.html

As example HTML page structure, the code below:

1. Sample html structure for header
<head>
    <meta charset="utf-8">
    <title>Tekno - Forum</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- Required -->
    <link href="assets/css/style.css" rel="stylesheet" type="text/css" media="screen">

CSS Files & Structure

  1. style.css - style css files (you can edit color, font, etc from this css)
  2. style-ie8.css - this style is condition for costume style on internet explorer 8
  3. bootstrap.css - css by Twitter
  4. bootstrap-responsive.css - css by Twitter for mobile version
